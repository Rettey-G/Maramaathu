## Production Notes

### Visual Style
- Clean, modern UI
- Maldives-inspired colors (blue, turquoise, white)
- Real screenshots from app
- Happy faces (diverse, local)

### Music
- Upbeat, tech-friendly
- 120-130 BPM
- Build excitement towards end

### Voiceover Options
1. **Professional Dhivehi** (recommended for local market)
2. **English with Dhivehi accent**
3. **Text-only with music** (for silent viewing)

### Required Assets
- [ ] App screen recordings (both customer & worker flows)
- [ ] Logo animations
- [ ] Stock footage of workers (plumber, electrician, AC tech)
- [ ] Happy customer testimonials (if available)
- [ ] QR code graphic
